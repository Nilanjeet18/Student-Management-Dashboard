# Student Management Dashboard

A web application to manage student records efficiently.

## Features

* Add new students
* Dashboard
* Delete student records
* Search students

## Tech Stack

* React.js
* Vite
* JavaScript
* CSS

```bash
cd Student-Management-Dashboard

npm install

npm run dev
```
